Зависимости устанавливаются через селект на странице контроля справочников.
Все можно спокойно создать через интерфейс,но если хотите вруную что-то добавить обновляйте базу данных и в папке с проектом в консоли python manage.py makemigrations и далее python manage.py migrate
Чтобы запустить проект - просто вбить в консоль в папке где находится проект python manage.py runserver
